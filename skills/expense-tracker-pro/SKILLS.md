# Pelacak Pengeluaran Pro

Lacak pengeluaran kamu lewat obrolan biasa. Gak perlu aplikasi atau spreadsheet — cukup bilang ke Clawd apa yang kamu belanjakan.

## Apa yang bisa dilakukan

Mencatat pengeluaran dari bahasa sehari-hari ("Saya menghabiskan Rp 45.000 untuk bahan makanan"), mengkategorikan otomatis, melacak terhadap anggaran, dan memberi ringkasan pengeluaran kapan pun kamu minta. Semua data disimpan secara lokal di perangkat kamu.

## Cara pakai

**Catat pengeluaran:**

```
"Saya menghabiskan Rp 23.500 untuk makan siang"
"Rp 150.000 untuk tagihan listrik"
"Kopi Rp 4.750"
```

**Cek pengeluaran:**

```
"Berapa pengeluaran saya minggu ini?"
"Tunjukkan pengeluaran makanan saya bulan ini"
"Apakah saya sudah melebihi anggaran hiburan?"
```

**Atur anggaran:**

```
"Tentukan anggaran bahan makanan Rp 400.000 per bulan"
"Atur anggaran Rp 100.000 untuk hiburan"
```

**Dapatkan laporan:**

```
"Ringkasan pengeluaran bulanan"
"Bandingkan pengeluaran dengan bulan lalu"
"Ekspor pengeluaran ke CSV"
```

## Kategori

Dideteksi otomatis dari konteks:

- Makanan & Minuman
- Transportasi
- Tagihan
- Hiburan
- Belanja
- Kesehatan
- Langganan
- Lain-lain

Misal tentukan kategori: "Menghabiskan Rp 50.000 untuk nasi goreng, kategori: Makanan"

## Tips

- Cantumkan jumlah dengan jelas supaya pencatatan akurat
- Tulis "berulang" untuk langganan: "Rp 15.000 Netflix, berulang tiap bulan"
- Tanyakan "tren pengeluaran" untuk melihat pola dari waktu ke waktu
- Semua data tersimpan secara lokal di perangkat kamu
